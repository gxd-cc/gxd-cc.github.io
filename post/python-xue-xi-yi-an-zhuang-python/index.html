Python学习（一）：安装Python

2020-04-19
  1. 首先从官网获取Python安装包。
  2. 打开下载好的安装包，首先勾选界面下面的Add Python 3.8 to PATH，然后选择Customize installtion（自定义安装位置）。
  3. 在新页面中直接点击Next即可。
  4. 在Advanced Options页面中选择自定义安装位置，然后点击Install。
  5. 等待安装结束后，界面出现Setup was successful则表示安装成功，此时点击Close按钮关闭安装程序，然后打开命令行界面，输入python，查看到下图的界面即表示Python环境变量配置成功。
